Tutorial Atelier Android

Installations

  1. Installer Virtual box: VirtualBox-4.3.12-93733-Win.exe
  2. Si l'installation demande de redémarrer, redémarrer.
  3. Double cliquer sur Mint32.ova : ça lance une machine virtuelle
  4. Une fois le système d'exploitation lancé, le mot de passe est "angie"

Lancements

  1. Sur le bureau, double cliquer sur [TODO icone] pour lancer
  2. Attendez la fin du lancement de l'emulateur de téléphone
  3. Sur le bureau, double cliqur sur [TODO img] pour lancer l'interface de développement
  4. Une fois l'interface lancée, cliquez sur [TODO img] pour lancer l'application Android sur le telephone
Remarque: l'emulateur de téléphone étant assez long a se lancer, il est conseillé de ne pas le fermer.

Description de l'environnement de développement

Atelier

L'environnement de développement est prêt, on entre maintenant dans le vif du sujet : faire une application Android.
Vous avez a disposition un squelette d'un jeu de mémory: c'est un jeu ou l'on retourne les cartes 2 par 2 et ou il faut retrouver les paires. Une fois toutes les paires trouvées, c'est gagné.

Ajouter deux cartes

Ajouter la regle du jeu

La premiere carte retounée, doit rester retournée jusqu'a ce qu'une deuxieme carte soit retournée. Lorsque les deux cartes sont sur retounée, si elles sont identiques alors il faut les garder de face, sinon il faut les remettre de dos. Le joueur a gagné lorsque toutes les cartes sont du coté face.
